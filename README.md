# vfox-squarectl

[![CI](https://github.com/jbox-web/vfox-squarectl/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-squarectl/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [squarectl](https://github.com/jbox-web/squarectl)

## Install

```
mise plugin install squarectl https://github.com/jbox-web/vfox-squarectl.git
```
